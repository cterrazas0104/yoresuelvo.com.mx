Presentación

El Centro de Estudios Superiores del Bajío (CESBA Campus Querétaro) invita a estudiantes, egresados y profesionistas a participar en este diplomado de vanguardia.

Actualmente, las organizaciones demandan aplicaciones hiperconectadas, capaces de procesar datos en tiempo real, integrarse con modelos de Inteligencia Artificial Generativa y operar con disponibilidad 24/7 sin la necesidad de mantener costosos servidores físicos o infraestructura compleja.

Concepto Clave

¿Qué es la Arquitectura Serverless e IA?

Es un paradigma de desarrollo donde los programadores construyen aplicaciones completas reutilizando servicios administrados en la nube.

GitHub Cloudflare Apps Script Gemini API

Esto permite desplegar plataformas escalables, seguras y protegidas con SSL en cuestión de minutos y a costo cero de servidor, acelerando drásticamente el tiempo de salida al mercado (Time-to-Market).

Estudiantes y Egresados

De la Licenciatura en Sistemas Computacionales e ingenierías afines interesados en fortalecer su portafolio profesional para procesos de titulación o inserción laboral.

Desarrolladores Web

Programadores que deseen actualizarse en arquitecturas de backend sin servidor, integración de microservicios e Inteligencia Artificial.

Consultores y Emprendedores

Tecnológicos que busquen crear productos digitales (SaaS) y herramientas de automatización empresarial minimizando costos operativos.

Objetivo General

Proporcionar a los participantes las competencias, metodologías y herramientas prácticas necesarias para diseñar, construir y desplegar aplicaciones web Full-Stack Serverless, integrando Bases de Datos en la nube, APIs de Inteligencia Artificial (Gemini API) y automatización de notificaciones (Webhooks), bajo esquemas avanzados de ciberseguridad y protección de dominios con Cloudflare.

Objetivos Específicos

Lo que serás capaz de hacer al finalizar

Flujo de Trabajo Moderno

Utilizar Git y GitHub como motor de control de versiones y despliegue continuo (CI/CD).

Infraestructura y Ciberseguridad

Configurar registros DNS, delegación de servidores, certificados SSL/TLS y capas de protección WAF/DDoS con Cloudflare.

Microservicios Serverless

Construir backends ligeros con Google Apps Script y APIs RESTful operando sobre Google Sheets como motor de datos.

Automatización Webhook

Conectar flujos de datos con plataformas de mensajería instantánea (CallMeBot / WhatsApp / Telegram).

Integración de IA Generativa

Consumir la API de Google Gemini mediante JavaScript para enriquecer la toma de decisiones y UX en tiempo real.

Temario

Contenido Curricular

Un viaje de aprendizaje estructurado en 6 módulos intensivos, diseñados para llevarte de los fundamentos al despliegue de software con Inteligencia Artificial.

01

Control de Versiones y Despliegue Continuo (CI/CD)

  • Fundamentos de Git y arquitectura de repositorios en GitHub.
  • Automatización de despliegues web con GitHub Pages / Cloudflare Pages.
  • Manejo de variables de entorno y seguridad en código fuente.
02

Infraestructura Web, DNS y Ciberseguridad Cloudflare

  • Anatomía de dominios web (TLDs, registradores, zonificación).
  • Gestión avanzada de registros DNS y configuración de Cloudflare (SSL/TLS).
  • Ciberseguridad: Firewall WAF, reglas de acceso y mitigación DDoS.
03

Backend Serverless y Arquitectura de APIs REST

  • Creación de endpoints doGet() y doPost() mediante Google Apps Script.
  • Google Sheets como motor de persistencia de datos relacional.
  • Control de concurrencia, LockService y sanitización de datos.
04

Webhooks, Event-Driven Architecture y Push

  • Arquitectura orientada a eventos.
  • Integración de Webhooks para procesamiento en segundo plano.
  • Notificaciones transaccionales a móviles (WhatsApp / Telegram API).
05

Integración de Inteligencia Artificial (Gemini API)

  • Consumo de APIs de IA utilizando JavaScript asíncrono.
  • Prompt Engineering: formateo de respuestas JSON desde la IA.
  • Automatización de análisis de datos e interacciones inteligentes.
06

Proyecto Integrador Full-Stack Serverless

  • Ensamblado final de la aplicación web completa.
  • Pruebas de estrés, optimización de velocidad de carga y UX.
  • Presentación y publicación oficial bajo dominio propio.
