Presentación

El Centro de Estudios Superiores del Bajío (CESBA Campus Querétaro) invita a estudiantes, egresados y profesionistas a participar en este diplomado de vanguardia.

Actualmente, las organizaciones demandan aplicaciones hiperconectadas, capaces de procesar datos en tiempo real, integrarse con modelos de Inteligencia Artificial Generativa y operar con disponibilidad 24/7 sin la necesidad de mantener costosos servidores físicos o infraestructura compleja.

Concepto Clave

¿Qué es la Arquitectura Serverless e IA?

Es un paradigma de desarrollo donde los programadores construyen aplicaciones completas reutilizando servicios administrados en la nube.

GitHub Cloudflare Apps Script Gemini API

Esto permite desplegar plataformas escalables, seguras y protegidas con SSL en cuestión de minutos y a costo cero de servidor, acelerando drásticamente el tiempo de salida al mercado.

¿A quién va dirigido este Diplomado?

Estudiantes y Egresados

De la Licenciatura en Sistemas Computacionales e ingenierías afines interesados en fortalecer su portafolio profesional para procesos de titulación o inserción laboral.

Desarrolladores Web

Programadores que deseen actualizarse en arquitecturas de backend sin servidor, integración de microservicios e Inteligencia Artificial.

Consultores y Emprendedores

Tecnológicos que busquen crear productos digitales (SaaS) y herramientas de automatización empresarial minimizando costos operativos.

Objetivo General

Proporcionar a los participantes las competencias, metodologías y herramientas prácticas necesarias para diseñar, construir y desplegar aplicaciones web Full-Stack Serverless, integrando Bases de Datos en la nube, APIs de Inteligencia Artificial (Gemini API) y automatización de notificaciones (Webhooks), bajo esquemas avanzados de ciberseguridad y protección de dominios con Cloudflare.

Objetivos Específicos

Lo que serás capaz de hacer al finalizar

Flujo de Trabajo Moderno

Utilizar Git y GitHub como motor de control de versiones y despliegue continuo (CI/CD).

Infraestructura y Ciberseguridad

Configurar registros DNS, certificados SSL/TLS y capas de protección WAF/DDoS con Cloudflare.

Microservicios Serverless

Construir backends ligeros con Google Apps Script y APIs RESTful operando sobre Google Sheets.

Automatización Webhook

Conectar flujos de datos con plataformas de mensajería instantánea (CallMeBot / WhatsApp).

Integración de IA Generativa

Consumir la API de Google Gemini mediante JavaScript para enriquecer la toma de decisiones.

Temario

Contenido Curricular

Un viaje de aprendizaje estructurado en 6 módulos intensivos.

01

Control de Versiones y Despliegue Continuo (CI/CD)

  • Fundamentos de Git y GitHub.
  • Despliegues con Cloudflare Pages.
  • Seguridad en código fuente.
02

Infraestructura Web, DNS y Ciberseguridad

  • Anatomía de dominios y DNS.
  • Certificados SSL/TLS.
  • Firewall WAF y mitigación DDoS.
03

Backend Serverless y APIs REST

  • Endpoints con Google Apps Script.
  • Google Sheets como base de datos.
  • Control de concurrencia y LockService.
04

Webhooks y Arquitectura Orientada a Eventos

  • Procesamiento en segundo plano.
  • Notificaciones transaccionales.
  • Integración con WhatsApp API.
05

Inteligencia Artificial (Gemini API)

  • Consumo de APIs con JavaScript.
  • Prompt Engineering y JSON.
  • UX inteligente en tiempo real.
06

Proyecto Integrador Full-Stack

  • Ensamblado final de la aplicación.
  • Pruebas de rendimiento y UX.
  • Publicación oficial con dominio propio.
