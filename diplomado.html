Presentación

El Centro de Estudios Superiores del Bajío (CESBA Campus Querétaro) invita a estudiantes, egresados y profesionistas a participar en este diplomado de vanguardia.

Actualmente, las organizaciones demandan aplicaciones hiperconectadas, capaces de procesar datos en tiempo real, integrarse con modelos de Inteligencia Artificial Generativa y operar con disponibilidad 24/7 sin la necesidad de mantener costosos servidores físicos.

Concepto Clave

¿Qué es la Arquitectura Serverless e IA?

Es un paradigma de desarrollo donde los programadores construyen aplicaciones completas reutilizando servicios administrados en la nube.

GitHub Cloudflare Apps Script Gemini API

Esto permite desplegar plataformas escalables, seguras y protegidas con SSL en cuestión de minutos y a costo cero de servidor.

¿A quién va dirigido este Diplomado?

Estudiantes y Egresados

De la Licenciatura en Sistemas Computacionales e ingenierías afines interesados en fortalecer su portafolio profesional.

Desarrolladores Web

Programadores que deseen actualizarse en arquitecturas de backend sin servidor e Inteligencia Artificial.

Consultores y Emprendedores

Tecnológicos que busquen crear productos digitales (SaaS) minimizando costos operativos.

Objetivo General

Proporcionar a los participantes las competencias, metodologías y herramientas prácticas necesarias para diseñar, construir y desplegar aplicaciones web Full-Stack Serverless, integrando Bases de Datos en la nube, APIs de Inteligencia Artificial (Gemini API) y Cloudflare.

Objetivos Específicos

Lo que serás capaz de hacer al finalizar

1. Flujo Moderno

Control de versiones con Git/GitHub y despliegue continuo (CI/CD).

2. Ciberseguridad

Gestión DNS, certificados SSL/TLS y capas WAF con Cloudflare.

3. Microservicios

Backends ligeros con Google Apps Script y persistencia en Google Sheets.

4. Webhooks & Push

Notificaciones transaccionales automáticas a WhatsApp y Telegram.

5. IA Generativa

Integración asíncrona con Gemini API y Prompt Engineering.

Temario

Contenido Curricular

ESTRUCTURADO EN 6 MÓDULOS INTENSIVOS PRACTICOS

01

Control de Versiones y CI/CD

  • • Fundamentos de Git y arquitectura en GitHub.
  • • Despliegues automáticos con Cloudflare Pages.
  • • Variables de entorno y seguridad de código.
02

Infraestructura y Cloudflare

  • • Zonificación DNS y registros de dominio.
  • • Certificados SSL/TLS gratuitos.
  • • Reglas WAF y protección anti-DDoS.
03

Backend Serverless & APIs

  • • Endpoints doGet() y doPost() con Apps Script.
  • • Google Sheets como Base de Datos.
  • • Control de concurrencia con LockService.
04

Webhooks & Event-Driven

  • • Procesamiento asíncrono en segundo plano.
  • • Integración de Webhooks.
  • • Notificaciones instantáneas por WhatsApp.
05

Integración de IA (Gemini API)

  • • Consumo de API mediante JS asíncrono.
  • • Prompt Engineering para respuestas JSON.
  • • Automatización de tareas con IA.
06

Proyecto Integrador Full-Stack

  • • Integración de arquitectura completa.
  • • Pruebas de rendimiento y UX.
  • • Despliegue oficial con dominio propio.
